# Quick Memo

一个轻量化的本地备忘录工具。置顶小窗口快速记录，Web 管理页统一查看。

约等于带了管理页面和窗口置顶的记事本，这恰是我的需求。

## 功能

- **置顶小窗口** — 始终显示在其他窗口上方，随时快速输入
- **富文本编辑** — 支持加粗、标题、列表、代码块、图片粘贴
- **分类与标签** — 预设分类 + 自由标签，多维度组织备忘录
- **Web 管理页** — 搜索、按分类/标签过滤、编辑、删除、置顶
- **全局快捷键** — Ctrl+Shift+M 呼出/隐藏窗口
- **图片支持** — 工具栏选择或剪贴板粘贴，自动上传存储
- **设置页面** — 管理分类、自定义快捷键

## 快速开始

### 安装依赖

```bash
pip install -r requirements.txt
```

### 启动

```bash
python main.py
```

启动后会：
1. 初始化 SQLite 数据库
2. 启动本地服务 (`http://127.0.0.1:18080`)
3. 弹出置顶的 Quick Memo 小窗口

## 使用

### 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Shift+M` | 显示/隐藏备忘录窗口 |
| `Ctrl+Shift+Q` | 退出应用 |
| `Ctrl+Enter` | 在编辑器中快速保存 |

快捷键可在管理页面的 Settings 中自定义（修改后需重启生效）。

### 编辑器窗口

- 顶部工具栏：加粗、斜体、标题、列表、代码块、图片
- 中间区域：富文本编辑，支持粘贴图片
- 底部栏：分类选择 + 标签输入 + 保存按钮
- 标签输入：按空格添加，点 x 删除

### 管理页面

浏览器打开 `http://127.0.0.1:18080`：

- 按关键词搜索
- 按分类、标签过滤
- 按时间排序
- 编辑、删除、置顶备忘录
- 点击右上角 Settings 管理分类和快捷键

### 关闭窗口

点击关闭按钮只会隐藏窗口，不会退出应用。使用 `Ctrl+Shift+Q` 或在管理页面退出。

## 项目结构

```
memo-app/
├── main.py              # 入口
├── app.py               # FastAPI 后端
├── desktop.py           # pywebview 置顶窗口
├── db.py                # SQLite 数据库
├── hotkey.py            # 全局快捷键
├── static/
│   ├── css/             # 样式
│   ├── js/              # 前端逻辑
│   └── vendor/          # Quill.js
├── templates/           # HTML 页面
├── uploads/             # 图片存储
├── memos.db             # 数据库文件（自动生成）
└── requirements.txt     # Python 依赖
```

## 技术栈

- **桌面窗口**: pywebview
- **后端**: FastAPI + Uvicorn
- **数据库**: SQLite (WAL 模式)
- **富文本编辑**: Quill.js
- **全局快捷键**: keyboard
